#### Using NodeJS (Windows and Linux)

#### Requirements

- Upstash Redis (Optional!!) [Find out how to create a redis database here](https://docs.upstash.com/redis), it's really simple.
- NodeJS Download [here](https://nodejs.org/en)
- NPM (Automatically installed when installing NodeJS)
- Discord Bot [Create one here](https://discord.com/developers/)
- Cloudflare R2 Bucket / S3 Bucket (Optional!!) [Find out how to create one here](https://developers.cloudflare.com/r2/get-started/)

#### Installation

- Download the repository and unzip it.
- Install NodeJS and NPM. (https://nodejs.org/en/download/)
- Rename .env.example to .env and fill in the required values. (Requirements can be found at the top of this page)
- Run "install_packages.bat"
- Run "start.bat"

Momentum should now be up and running!

If you encounter any errors during the installation, please join our Discord Server at https://discord.gg/HuYyFYfwNp and ask for help in the #help channel.
Any errors shown in console can be copied and pasted there, and the bot will already answer the most common questions automatically. If you still cannot get it to start up correctly, please refer to @Zetax.
